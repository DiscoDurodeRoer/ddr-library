¿Qué es la biblioteca de componentes DdrLibrary?
Es un conjunto de componentes/servicios/pipes/directivas reutilizables para tus proyectos.
Como instalar la biblioteca DdrLibrary
Para instalar la biblioteca DdrLibrary, ejecutar el siguiente comando: 
npm i ddr-library
¿Que debemos añadir en el fichero angular.json?
Añadiremos lo siguiente en assets: 
{ "glob": "**/*", "input": "node_modules/ddr-library/resources/img", "output": "resources/img/"}
Añadiremos lo siguiente en styles: 
"node_modules/bootstrap/dist/css/bootstrap.min.css",
"node_modules/font-awesome/css/font-awesome.min.css",
"node_modules/ddr-library/resources/scss/font.scss",
"node_modules/ddr-library/resources/scss/ddr-styles.scss",
"node_modules/ddr-library/resources/themes/theme-default.scss",
"node_modules/ddr-library/resources/themes/theme-dark.scss"
¿Que agregar en app.module?
Añadiremos lo siguiente en app.module: 
import {BrowserAnimationsModule} from '@angular/platform-browser/animations';
import { DdrLibraryModule } from 'ddr-library';

@NgModule({
  imports: [
    BrowserAnimationsModule,
    DdrLibraryModule
  ],
  bootstrap: [AppComponent]
})
export class AppModule { }
